USE Flag Symbols

The following symbols are used in the package views to indicate the status of USE flags across different versions:

  • ✓ : The USE flag is present in the ebuild.
  • ⊕ : The USE flag is present and enabled by default (+flag).
  • ⊖ : The USE flag is present and disabled by default (-flag).
  • ✗ : The USE flag is absent in the ebuild version.
