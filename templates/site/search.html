Search Documentation

Use the search box above to find packages. You can use plain text to search across names, descriptions, and use flags, or use specific field filters.

Syntax

  • Plain text: ollama - matches anywhere in the package name, description, etc.
  • Field filters: category:app-admin, license:MIT, arch:amd64, keyword:~amd64
  • Boolean Logic:
    • Implicit AND: app-admin ollama
    • Explicit OR: license:MIT OR license:BSD
    • Negation: -mask:hard or NOT mask:hard or !mask:hard
  • Grouping: (license:MIT OR license:BSD) AND arch:amd64
  • Sequences: 'system user - searches for an ordered sequence of words.

Supported Fields

  • overlay: Repository name
  • category: Package category
  • url: Package URL
  • arch: Architecture (e.g., amd64)
  • keyword: Raw keyword (e.g., ~amd64)
  • mask: Mask status (none, soft, hard)
  • version: Version comparison (e.g., v3.4.5, >v3.4.5, <v3.4.5)
  • license: License name
  • depends, rdepends: Packages it depends on
  • depended, rdepended: Packages that depend on it
  • manifestfile: Files in the manifest
  • EAPI, slot, inherit, use

Examples

  • ollama
  • license:MIT AND arch:amd64
  • depends:dev-lang/go
  • rdepended:sys-libs/zlib
  • mask:none use:xorg
  • version:>v3.4.5
  • 'system user
